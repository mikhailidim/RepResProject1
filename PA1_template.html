Detailed examine show that we have 2304 records or 13.11% from overall 5 minutes inerval measurments. To fill gaps we may use rounded avergae steps for interval in the same month. The make better assumption we will use data for the same day of week. Let's implement our impute strategy with some calculations.

  1. Copy source dataset into new one with new day of week column.

    iactv<-cbind(activity,dow=weekdays(activity$date))

  2. As part of data preparation separate weekdays from weekends.

    iactv$weekpart <- sapply(iactv$dow,FUN = function(x){
                                            switch(as.character(x),
                                                   Sunday="Weekend",
                                                   Saturday="Weekend",
                                                   "Weekday")})

  3. Compute means for interval, month, and day of week combination.

    avglkp<-group_by(iactv,interval,mname=months(date),dow) %>% 
            filter(!is.na(steps)) %>%
            summarize(steps.mean=round(mean(steps,na.rm=T)))

  4. Helper function to lookup steps in average objects 

    impute<-function(x) { 
if (is.na(x$steps)) {
 as.integer(avglkp[avglkp$mname == months(x$date) & 
                           avglkp$interval == x$interval & 
                           avglkp$dow == x$dow,"steps.mean"])
} 
else {
 as.integer(x$steps)
}
}

  5. Populate missing measurments with average values.
